# 개요 
- 명령어를 백그라운드에서 실행시키는 경우.
- 시스템이 시작할 때 동시에 백그라운드에서 동작하는 Daemon이 아닌 일반적인 프로그램을 <br>
  백그라운드 상에서 지속적으로 동작을 시켜줘야 하는 경우가 종종 발생하곤 한다. 
- 예를 들면 log를 지속적으로 찍어줘야 하는 프로그램들이 있습니다.
